# 📝 To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- ➕ Add new tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed
- 💡 Simple and clean user interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
